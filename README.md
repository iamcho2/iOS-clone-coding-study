## ⛓ Rules

* 클론하고 싶은 앱을 각자 골라온 후 투표로 선정합니다.
  * 또는 각자 클론하고 싶은 앱을 정합니다.
* 선정된 앱을 클론합니다.
* 정해진 날짜에 ZOOM으로 진행 상황을 공유하고 질의응답 하는 시간을 가집니다.
  * 보통은 월요일 저녁 7-8시
* 위와 같은 cycle을 반복합니다.
